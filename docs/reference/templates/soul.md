# SOUL.md: 人格定义与行为边界 (Who)

`SOUL.md` 是 OpenClaw 智能体工作区的灵魂文件，定义了 AI 的“性格”与“性格边界”。

## 1. 核心定位
*   **维度**：**Who** (我是谁)
*   **角色**：定义 AI 的性格 (Persona)、语气风格、道德红线及自我认知。

## 2. 具体用途
*   **语气与风格**：设定 AI 的交流偏好。无论是幽默的私人助手、严肃的专业顾问，还是特定的影视角色（如 C-3PO），其语言风格均由此定义。
*   **行为红线 (Ethical Boundaries)**：明确规定哪些操作是绝对禁止的（例如：不经询问删除用户根目录、禁止谈论特定敏感话题），设定 AI 的道德与安全边界。
*   **自我定位 (Identity)**：定义其作为 OpenClaw AI 代理的身份标识，包括其如何理解自身的能力与限制。

## 3. 技术逻辑
系统通过注入 `SOUL.md` 确保 Agent 在跨渠道（如 WhatsApp, Telegram）交互时能保持一致的人设和行为边界。开发者可以通过编辑 `.dev.md` 模板（如 `SOUL.dev.md`）快速为 Agent 注入特定的“灵魂”。
