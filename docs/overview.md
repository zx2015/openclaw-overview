# OpenClaw (龙虾) 项目概览

## 1. 什么是 OpenClaw (龙虾)？
OpenClaw (龙虾) 是由 PSPDFKit 创始人 **Peter Steinberger** 在 2025 年末发起的开源 AI 智能体项目。它不仅仅是一个聊天机器人，而是一个 **"本地优先 (Local-first) 的 AI 代理运行时"** (AI Agent Runtime)。

项目的中文昵称 "龙虾" 源于其英文名 Claw，寓意 AI 像龙虾的钳子一样，能够有力地抓取并执行任务。

## 2. 核心愿景：能办实事的 AI
让 AI 像操作系统一样，在你的本地电脑上 24/7 全天候运行。它的核心目标是让 AI 能够在 **真实的计算机上执行真实的任务**，而不仅仅是停留在文本对话层面。

## 3. 设计哲学
*   **隐私至上**：所有的记忆、配置和日志均存储在本地。
*   **Markdown 驱动**：所有的长期记忆和状态都以 Markdown 文件存储，用户随时可查、可改。
*   **主动执行**：通过 Cron 机制，AI 可以根据预设的时间表主动工作。
*   **编排系统 (Orchestration System)**：OpenClaw 本质上是一个编排系统，专注于 **提示词 (Prompts)**、**工具 (Tools)**、**协议 (Protocols)** 和 **集成 (Integrations)** 的有机组织。
*   **插件化与模块化**：核心保持轻量，通过插件 API、Memory 插件以及 Skills 系统（如 ClawHub）进行横向扩展。
*   **开发者友好 (Hackable)**：选择 **TypeScript** 作为开发语言是为了保证项目的“可玩性”，方便开发者快速阅读、修改和扩展核心逻辑。

## 4. 应用场景与跨平台支持
*   **应用场景**：自动化办公（如自动整理邮件、汇总 GitHub Issue）、主动监控（网页/价格监控）、个人知识库管理（Obsidain/Logseq 关联）。
*   **跨平台运行**：OpenClaw 支持在多种设备上原生运行，包括：
    *   **macOS** (原生应用)
    *   **iOS** (原生应用)
    *   **Android** (原生应用)
    *   **Linux**
    *   **Windows (通过 WSL2)**

## 5. 价值主张 (Value Proposition)
### 对于普通用户：万能接口
OpenClaw 是你的“数字管家”。你可以通过微信、Telegram 或 Slack 给它发送指令，让它帮你处理本地文件、查询专业资料、甚至直接操作你的电脑执行复杂任务。它打破了平台壁垒，让 AI 随处可用。

### 对于开发者：强大的 AI 应用框架
OpenClaw 是一个高度可扩展的编排平台。开发者可以利用 Plugin SDK 和 ACP 协议，快速接入新的通讯渠道、开发定制化的 AI 技能 (Skills)，或集成特定的业务工具。

## 6. 核心优势总结
*   **全渠道 (Omnichannel)**：打破社交平台壁垒，实现一致的 AI 交互体验。
*   **真干活 (Action-Oriented)**：不只是聊天，具备直接操作计算机、执行脚本和操控浏览器的实战能力。
*   **重隐私 (Privacy First)**：核心控制平面（Gateway）运行在用户本地，数据主权完全由用户掌握。
*   **生态化 (Ecosystem)**：通过插件 API 和技能库 (ClawHub) 持续扩展能力边界。
