# OpenClaw AI 智能体 搜索摘要 (2026-03-10)

## 项目定位
OpenClaw 是一个由 Peter Steinberger 发起的开源项目，定位为 **"AI 代理的本地运行时"** 或 **"AI 操作系统"**。它旨在让 AI 智能体能够 24/7 全天候在本地运行，并具备主动执行任务的能力。

## 关键技术点
1.  **ReAct 推理循环**：核心架构采用 Reasoning + Acting 模式，使 AI 具备逻辑链条。
2.  **本地化存储**：弃用复杂数据库，使用 **Markdown 文件** 存储记忆，方便用户直接查看和 Git 备份。
3.  **主动性 (Proactivity)**：集成 Cron 任务，AI 可以主动发起行动，而非等待用户提问。
4.  **本地模型支持**：无缝对接 Ollama、LM Studio 等本地模型提供方。

## 主要组件
*   **Clawd**：核心后台守护进程。
*   **ClawHub**：技能/插件共享平台。
*   **Skills**：独立的功能单元，如文件管理、Web 搜索、邮件发送等。

---
*注：本项目已明确排除 OpenClaw 同名游戏引擎的内容，专注于 AI 智能体领域。*
